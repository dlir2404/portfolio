## Why Dual Boot?

Dual booting gives you the best of both worlds - you can keep your familiar Windows system while exploring the powerful features of Ubuntu Linux. It's perfect for:
- Learning Linux without commitment
- Development work that requires Linux
- Running software specific to each OS
- Maintaining Windows for gaming while using Linux for work

## Prerequisites

Before we begin, make sure you have:
- Windows 10/11 already installed
- At least 30GB of free disk space
- A USB drive (8GB or larger)
- Backup of important data
- Basic knowledge of BIOS/UEFI settings

## Step-by-Step Guide

### 1. Create Ubuntu Installation Media
First, we'll need to create a bootable USB drive:
1. Download Ubuntu from ubuntu.com
2. Download Rufus USB creator
3. Create bootable USB using Rufus

### 2. Prepare Your Windows System
- Backup important data
- Create free disk space by shrinking Windows partition
- Disable Fast Startup in Windows
- Disable Secure Boot in BIOS
